MEDIC Client Registry FHIR Reference Implementation

This is the landing page for the MARC-HI Client Registry reference implementation FHIR service. This registry "speaks" a variety of standards including:

  • HL7 Version 2.5 - PIX & PDQ Version 2
  • HL7 Version 3 - pan-Canadian Standards Client Registry Messages R02.04.02
  • HL7 Version 3 - IHE PIX & PDQ Version 3
  • FHIR - This resource

In order to use the FHIR service you will have to communicate with one of the implemented FHIR resources using the specification found here:

  • Profile Resource - Describes the profiles used on this client registry
  • ValueSet Resource - Contains a list of custom codes used on this registry
  • Patient Resource - Patients contained in the client registry
